# Expense Manager – খরচের হিসাব রাখার অ্যাপ

একটি সম্পূর্ণ Next.js অ্যাপ্লিকেশন যা আপনাকে একাধিক টেবিলে খরচের হিসাব রাখতে, ফিল্টার করতে, চার্ট দেখতে এবং Excel-এ ডাউনলোড করতে সাহায্য করে।

## ✨ ফিচারসমূহ

- **লগইন সিস্টেম** – ইউজার ও অ্যাডমিন রোল
- **অ্যাডমিন প্যানেল** – ইউজার অ্যাড ও ডিলিট
- **একাধিক টেবিল** – আলাদা আলাদা হিসাব রাখা যায়
- **ক্যাটাগরি** – খরচকে ৮টি ভাগে ভাগ করা যায়
- **ফিল্টার** – তারিখ ও ক্যাটাগরি অনুযায়ী খরচ দেখানো যায়
- **চার্ট** – ক্যাটাগরি ভিত্তিক খরচের বার চার্ট
- **ডার্ক/লাইট থিম** – ব্যবহারকারী পছন্দমতো পরিবর্তন
- **Excel ডাউনলোড** – ফিল্টার করা ডেটা সহ এক্সেল (বাংলা)
- **মোবাইল-ফ্রেন্ডলি** – রেসপন্সিভ ডিজাইন
